🔧 Predictive Modelling of Truck Air Pressure System Failure
MSc Dissertation · University of Kent · 2024
End-to-end machine learning pipeline to predict failures in truck Air Pressure
Systems (APS) using a real-world Scania fleet dataset of 60,000+ records across
170 features.
What I did:

Conducted comprehensive data cleaning, imputation and feature selection in
Python (Pandas, NumPy) — handling significant missing values across 170 variables
Applied and compared three classification algorithms: Logistic Regression,
Random Forest and Decision Tree
Addressed severe class imbalance using resampling techniques to improve
failure detection sensitivity and reduce costly false negatives
Evaluated performance using accuracy, precision, recall, F1-score and confusion
matrices — Random Forest delivered the strongest overall results
Produced clear visualisations using Matplotlib and Seaborn to communicate
findings to non-technical audiences

Key outcome: Demonstrated practical value of predictive analytics in reducing
unplanned downtime and maintenance costs in safety-critical fleet environments.
Python Scikit-learn Pandas NumPy Matplotlib Seaborn
